# Project Index

## Troquim Bot

Sistema de agendamento para salões de beleza via WhatsApp.

### Visão Geral
- **Tipo**: Aplicação Spring Boot
- **Linguagem**: Java
- **Arquitetura**: Hexagonal/Clean Architecture
- **Status**: MVP em desenvolvimento

### Módulos Principais
- **Conversation** - Motor de conversa WhatsApp
- **Appointment** - Agendamentos
- **Reservation** - Reservas temporárias
- **Customer** - Clientes
- **Professional** - Profissionais
- **Service** - Serviços
- **Availability** - Disponibilidades
- **Business** - Configurações do salão

### Tecnologias
- Spring Boot
- Spring Web
- Spring Test
- Jackson (JSON)
- RestTemplate (HTTP client)
